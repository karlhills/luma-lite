# Security Policy

## Supported Versions

We support the latest release only. Please update to the most recent version before reporting issues.

## Reporting a Vulnerability

If you discover a security issue, please email: support@84boxes.com

Please include:
- A clear description of the issue
- Steps to reproduce
- Impact assessment (if known)
- Any relevant logs or screenshots (redact secrets)

We will acknowledge reports as soon as possible and work on a fix.
